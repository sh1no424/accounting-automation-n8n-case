Проект: Автоматизация документооборота в 1с через n8n. Прием и обработка УПД.

Проект направлен на автоматизацию приема и обработку первичных документов в 1с.
Основной пользователь процесса - главный бухгалтер, который контролирует выполнение процесса, решает спорные моменты. 
До автоматизации главной проблемой были: скорость обработки, риск ошибок, дублирование документов.
Решение на базе n8n автоматически проверяет реквизиты, соответствие договору, номенклатуру, схождение сум НДС, заполнение строк 5a/5б. создает документ «Поступление товаров/услуг» на основании УПД в 1с, подставляет контрагента по ИНН, выбирает договор по номеру/наименованию, заполняет табличную часть, автоматически контролируя тип документа, отражение по счетам, дублирование документов по номенклатуре, по одному и тому же договору/периоду.
