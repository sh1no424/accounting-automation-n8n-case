# Маппинг данных
## Назначение документа
Цель - зафиксировать, какие поля извлекаются из входящего докуменнта, как они используются и приводятся в единый формат.

## Маппинг реквизитов документа
| Источник | Поле JSON | Использование | Описание|
| --- | --- | --- | --- |
| Источник поступления| source | проверка и аналитика | Показывает откуда поступил документ |
| Тип документа | document.type | Проверка типа документа | Например УПД |
| Формат файла | document.format | выбор способа обработки | xml pdf image unknown |
| Номер УПД | document.number | Поиск дублей, создание документа в 1С | Номер входящего документа  |
| Дата УПД | document.date | Поиск дублей, создание документа в 1с | Дата составления документа |
| ID документа | document.external_id | Защита от повторной обработки | Используется только для документов, полученных из СБИС. |
| Имя файла | document.file_name | Логирование и диагностика | Используется для понимания какой файл обрабатываем |


## Маппинг поставщика
| Источник | Поле JSON | Использование | Описание|
| --- | --- | --- | --- |
| Наименование поставщика | supplier.name | Заполнение документа, уведомления | Наименование контрганета из УПД |
| ИНН поставщика | supplier.inn | Проверка в реестре ФНС, 1С | Основное поле для индентификации контрагента |
| КПП поставщика | supplier.kpp | Проверка в реестре ФНС, 1С | Для уточнения юр. лица |

## Маппинг покупателя
| Источник | Поле JSON | Использование | Описание|
| --- | --- | --- | --- |
| Наименование покупателя | buer.name | Поиск организации в 1С | Название организации, на которую выставлен УПД |
| ИНН покупателя | buyer.inn | Поиск организации в 1С | --- |
| КПП покупателя | buyer.kpp | Поиск организации в 1С | --- |

## Маппинг договора
| Источник | Поле JSON | Использование | Описание|
| --- | --- | --- | --- |
| Номер договора | contract.number | Поиск договора в 1С | Для сопоставления документа с договором |
| Дата договора | contract.date | Поиск договора в 1С | Для сопоставления документа с договором |
| Наимнование договора | contract.nam | Доп проверка | Используется, если номер и дата договора отсутствуют |

## Маппинг сумм и НДС
| Источник | Поле JSON | Использование | Описание|
| --- | --- | --- | --- |
| Сумма без НДС | amounts.sum_without_vat | Проверка математики НДС | Сумма документа до начисления НДС |
| Ставка НДС | amounts.vat_rate | Проверка допустимой ставки НДС | Например: 0, 5, 7, 10, 20, 22 или без НДС |
| Сумма НДС | amounts.vat_sum | Проверка математики НДС | Сумма налога по документу |
| Итоговая сумма | amounts.total_sum | Проверка математики НДС | Общая сумма документа с НДС |

## Маппинг строк 5а и 5б УПД
| Источник | Поле JSON | Использование | Описание|
| --- | --- | --- | --- |
| Строка 5а УПД | upd_fields.field_5a | Проверка корректности УПД | Содержит основание передачи или документ отгрузки |
| Строка 5б УПД | upd_fields.field_5b | Проверка авансовых счетов-фактур | Проверяется при наличии авансовых документов |

## Маппинг табличной части
| Источник | Поле JSON | Использование | Описание|
| --- | --- | --- | --- |
| Номер строки | items[].line_number | Сохранение порядка строк | номер позиции |
| Тип позиции | items[].type | Классификация номенклатуры | Возможные значения: goods, material, service, leasing |
| Наименование позиции | items[].name | Поиск номенклатуры или заполнение содержания | Исходное наименование |
| Количество | items[].quantity | Создание табличной части | Количество по строке документа |
| Единица измерения | items[].unit | Создание табличной части | Например: шт, усл., кг. |
| Цена | items[].price | Создание табличной части в 1С | Цена за единицу |
| Сумма без НДС | items[].sum_without_vat | Проверка сумм по строкам | Сумма строки без НДС |
| Ставка НДС | items[].vat_rate | Проверка НДС по строке | Ставка НДС для конкретной позиции |
| Сумма НДС | items[].vat_sum | Проверка НДС по строке | НДС по строке документа |
| Итоговая сумма | items[].total_sum | Проверка итогов и передача в 1С | Общая сумма строки |



## Маппинг классификации номенклатуры
| Источник | Поле JSON | Использование | Описание|
| --- | --- | --- | --- |
| Товар | goods | Должен быть найден в справочнике номенклатуры 1С | Передается найденная номенклатура |
| Материал | material | Должен быть найден в справочнике 1С или создан при отсутствии | Передается найденная или созданная номенклатура |
| Услуга | service | Если точное совпадение не найдено, используется обобщенная номенклатура "Услуги" | В номенклатуру передается "Услуги", исходное название сохраняется в содержании |
| Лизинг | leasing | Используется обобщенная номенклатура "Лизинг" | В номенклатуру передается "Лизинг", исходное название сохраняется в содержании |


## Маппинг правил поставщиков
| Поле JSON | Использование | Описание|
| --- | --- | --- |
| supplier_inn | ИНН поставщика, для которого применяется правило | 0000000000 |
| supplier_kpp | КПП поставщика, для которого применяется правило | 000000000 |
| settlement_account | Счет расчетов с поставщиком | 60.01 | 
| advance_account | Счет авансов. | 60.02 | 
| expense_account | Счет учета затрат или расходов | 26 | 
| vat_account | Счет учета НДС | 19.04 | 
| active | активность правила | true |
